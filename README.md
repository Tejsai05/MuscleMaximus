# MUSCLE MAXIMUS

A simple multi-page fitness-themed static site with landing, login, and signup pages.

## Overview
- **Landing page:** index.html
- **Authentication pages:** login.html, signup.html
- **Styles:** styles.css, login.css, signup.css
- **Assets:** /assets

## Run locally
The project is already running via a local static server.

- URL: http://localhost:5173
- Pages:
  - /index.html
  - /login.html
  - /signup.html

If you need to start it again:
1. Open a terminal.
2. Run: `python3 -m http.server 5173`
3. Visit http://localhost:5173
